# Keepalived User Guide

This guide walks through the concepts behind Keepalived, how to install it and how
to configure load balancing and high availability. It started life as the original
Sphinx user guide and has been refreshed for this site, with the configuration
details aligned to the current [keepalived.conf(5)](../keepalived-conf.md) reference.

Some operational examples remain historical illustrations. For the exact keywords
and defaults of your release, always rely on
[keepalived.conf(5)](../keepalived-conf.md), and for current packages see the
[Download](../../download.md) page.
